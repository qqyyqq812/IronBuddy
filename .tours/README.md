# Code Tours

这个目录存放项目的 Code Tour 导览文件（`.tour` JSON 文件）。

## 使用方式

1. 在 VS Code 中安装 **CodeTour** 扩展
2. 打开任意 `.tour` 文件，VS Code 会自动提示"Start Tour"
3. 按步骤逐个理解代码

## 命名规范

- `cc-改动讲解-YYYY-MM-DD.tour` — CC 改动的讲解（AG 生成）
- `architecture-overview.tour` — 项目架构导览
- `[topic]-[depth].tour` — 其他导览

## 由 AG 维护

AG 会根据用户的需求，生成对应的 Code Tour 文件供讲解。
