# ALSA 音频硬件增益调节指南 (麦克风静音排查篇)

在边缘计算板卡（如 RK3399ProX）或 Linux 终端环境下，麦克风灵敏度低或完全“空耳”往往是因为 ALSA 底层声卡的 `Capture`（录音参数）默认处于出厂的低增益或静音锁定状态。

本指南提供**图形化界面**和**单行指令**两种方式，教你手动排查并拉满麦克风。

---

## 方法一：使用 `alsamixer` 图形仪表板 (推荐)

这是最安全、最全局的排错手段。当你不确定当前声卡有什么控制杆时，可以进入这个界面用方向键可视化调节。

1. **进入开发板终端**（你可以通过 SSH 登入）。
2. **输入 `alsamixer` 并回车**。
3. **选择声卡【关键步】**：
   - 默认打开的是主板自身的编解码器（RK809）。但这不一定是我们正在用的 USB 摄像头麦克风。
   - 按下按键 **`F6`** (如果键盘的 F6 被占用，可以输入字母 `s`)，弹出一个列表。
   - 用方向键选择 `HD 720P Webcam`（我们的 USB 摄像头），按回车确认。
4. **切换到录音面板**：
   - 按下按键 **`F4`** 进入 `Capture` 面板（默认 F3 是播音，F4 是录音）。
5. **拉满你的增益**：
   - 使用 `←` `→` 方向键选中名为 `Mic`（或 `Capture`）的柱状条。
   - 疯狂按键盘的 **`↑` (上方向键)**，直到柱体升至 100。
   - ⚠️ *注意：如果柱状条底部写着 `[MM]`（带方框的 M），代表被静音了；按下字母键盘的旁边的 **`M`** 或 **空格键** 接触静音，直到变成绿色的 `[00]`。*
6. 按下 `Esc` 退出界面。

---

## 方法二：直接注射一行终端命令 (快速)

如果你已经确信目前挂接的是哪个硬件设备号，比如我们目前挂接的是序号卡 `2`（或者是用 `amixer -c Webcam`），你可以直接复制这行指令给板子强行打满鸡血：

```bash
# 解除你的 USB Webcam 的主 Mic 硬件静音，并将采音放大器开到 100% 满压
amixer -c 2 sset 'Mic' 100% unmute
```

---

## 方法三：固化设定 (防止重启丢失)

边缘板卡在每次断电并重新热拔插 Linux USB 总线时，有时会丢失你刚刚辛苦调好的 `100%` 设置，恢复成出厂极低的音质。我们可以将你当下的最佳组合锁进硬盘：

```bash
sudo alsactl store
```

### 小知识：声卡排查三板斧
1. `arecord -l`：列出当前系统所有**录音设备**及对应序号（如 `card 2`）。
2. `amixer -c 2 scontents`：翻找这块 `card 2` 肚子里的所有旋钮。
3. `arecord -D hw:2,0 -f S16_LE -r 16000 -c 1 /tmp/test.wav`：直接向 `hw:2,0` 要两声看看听不听得见。
