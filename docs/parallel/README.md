# 🤖 IronBuddy 并行开发总署 (Parallel Working Directory)

**状态：【🟢 ON 火力全开】**

欢迎进入高通量代码推进中心。本目录专用于多 Agent 在物理隔离下的**并行自治与开发迭代**。所有的代码、配置变更通过各个 `agent_` 的独立空间规划与缓冲，最后由 Director 执行审阅合并。

## 各部门现存长效总监
* **Agent 1**: 语音通讯突破口 (负责彻底修缮失联的拟态教练 TTS/ASR 框架)
* **Agent 2**: 视觉 NPU 推断优化组 (主刀 RTMPose 的换血接入)
* **Agent 3**: 绝杀算法融合组 (挑战视觉骨点+频域肌电的 Late Feature Fusion 网络)

## 查阅手册 (Director Rules)
- 要发动作战指令分流：召唤 Director 执行 `/para-dispatch`。
- 要收拢代码强制检查：召唤 Director 执行 `/para-join`。
- 单独与某一位 Agent “一对一”代码审阅：切管唤醒特定的工作流模型，通过左侧对应文件夹的 `assignment.md` 唤起记忆。
