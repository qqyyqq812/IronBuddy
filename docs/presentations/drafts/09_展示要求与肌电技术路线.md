# 第九章：IronBuddy V3.0 展示呈现要求与肌电技术路线概览

本章规定了 IronBuddy V3.0 在最终阶段（2026 ISEE-瑞芯微移动创新竞赛）的现场实机答辩要求，并详细记录了作为 V3.0 核心突破的表面肌电（sEMG）多模态技术实现路线。

## 1. 核心呈现目标与技术价值

在 V2.X 版本中，系统仅依赖纯视觉（Pose Estimation）进行健身姿态捕捉。在深蹲、臀桥等复杂动作中，存在自遮挡以及无法监测肌肉实际发力内况等痛点。V3.0 通过引入表面肌电（sEMG）传感器作为底层多模态数据源，向评委展示了“视-感融合”（视觉测姿态 + 肌电测发力）的高维 AI 教练能力。

现场展示需要重点体现这一技术价值，明确系统能够实现跨设备的无缝软硬件联动。

## 2. 现场实机答辩具体要求

实机答辩应遵循务实原则，重点在于证明技术路线可行性与数据流完整性。

### 动作要求
*   **二头弯举 (Bicep Curl)**：展示手臂动作追踪，由于需要捕捉发力情况，肌电传感器需紧贴于肱二头肌。
*   **深蹲/臀桥 (Squat/Glute Bridge)**：展示下肢与核心群动作追踪，肌电传感器需贴敷于股四头肌或臀大肌。

> [!IMPORTANT]
> **关键约束**：传感器硬件形态必须支持灵活吸附于身体任意大肌群。

### 物理形态要求
实机展示中接受适度的硬件裸露，以此体现底层链路的开发深度与原始的工业感。线路保持整洁有序（使用扎带或理线夹规划）即可，重在逻辑清晰、链路可循。

## 3. 肌电系统技术路线 (模拟 EMG + ESP32 桥接)

综合考量后，本项目确定采用「模拟 EMG 模块 + ESP32 BLE 桥接」路线。该方案通过无线蓝牙传输，彻底切断了受试者人体与主测试台（RK3399ProX）之间的物理有线束缚。

### 核心模块
系统由收纳在穿戴背囊内的穿戴主机及外部电极贴片组成：
1.  **终端采集 (模拟 sEMG)**：选定 AD8232 单导模块，利用内置仪表放大器捕捉微弱的差分肌电信号，输出 0~3.3V 模拟电压包络。
2.  **中枢处理 (ESP32 WROOM-32)**：作为穿戴大脑，利用内部 ADC 连续读取 4 个 EMG 模块信号进行同步采样汇总，并通过蓝牙协议对外广播。
3.  **系统供电**：采用 3.7V 小体积聚合物锂电池（带 JST 接口）为全体模块统一供电，保证无相互干扰与系统共地稳定。

### 硬件避坑与防冲突布线
> [!WARNING]
> 必须避免使用引脚资源冲突的 ESP32-C3 等其他型号，定死采购标准版 `ESP32-WROOM-32` 开发板。

ESP32 的 ADC2 通道底层硬件与 Wi-Fi /蓝牙控制器发生强复用。一旦使用蓝牙会导致连带的 ADC2 采样模块崩溃。**必须**将 4 路肌电信号全部接入独立安全的 **ADC1 通道**：
*   通道 1 (肱二头肌)：GPIO 34 (`ADC1_CH6`)
*   通道 2 (股四头肌)：GPIO 35 (`ADC1_CH7`)
*   通道 3 (臀大肌)：GPIO 32 (`ADC1_CH4`)
*   通道 4 (腘绳肌)：GPIO 33 (`ADC1_CH5`)

### 主线通信落盘流程
ESP32 工作在 BLE GATT Server 模式。它将高频采集的数据打包，并通过特征值 (Characteristic) 以 `Notify`（通知）方式持续向外界广播。
接收端的 RK3399ProX 作为 BLE Client，运行基于 Python 的 `bleak` 异步通信库并发爬取底层蓝牙特征值数据，并将原始序列实时写入系统的高速内存共享区 (`/dev/shm`)，最终供核心主循环引擎拉取调用。

## 4. 穿戴形态一致性与材料购置规划

实机场地的空间与受试者大幅度运动对系统的封装稳定度提出了严苛挑战。在放弃繁重工业外壳的前提下，本项目对软穿戴体感与走线系统进行了以下精细化布置：

### 核心辅具包裹：运动小背包
购置一款紧身、轻量透气的**运动小背包（或越野跑战术背心）**作为中枢物理容器，替代传统的口袋或硬绑带。
1. **收容与物理防损**：其内部分层可以完美无缝装载 1 块 ESP32、4 块 AD8232 集成板及锂供电系统，隔绝剧烈运动时的汗液，并对脆弱的焊接针脚起到静态缓冲保护。
2. **人体工学贴合**：背包系统高度贴合躯干背部。在激烈的大幅度深蹲或快速冲刺弯举时，背包主体也不会发生抛升或移位，避免影响测试者的运动平衡学表现。
3. **线缆约束与导流（辅助固线）**：背负系统的宽体肩带以及腰环可作为现成的“走线暗槽”。传感器的长段杜邦线与导联线均可以通过肩带与安全卡扣进行缠绕与顺线收纳，杜绝挂绊隐患。

### 传感引流线路配置
结合无纺布医用电极贴片，线缆长度采取“近身长导，下颚短导”逻辑：
*   **上肢游线**：手臂肱二头肌部分使用 **60cm** 较长三芯扣式导联线。为手部从全伸展到极限弯曲提供充足形变冗余空间，多余线材依靠背带隐藏。
*   **下颚走线**：大腿（股直肌）与臀大肌采集路线由于靠近背包下摆底座，统一采用 **30cm** 短款导电线材跨栏下沉，强制限制悬吊幅度。

> 以下为仿真环境证实图：即便是经历在站立、极限深蹲以及受力弯举这三种幅度差异极具代表性的动作切变中，受试者的线束宽裕度与背负重力都表现均匀，电极无拉扯滑脱。
