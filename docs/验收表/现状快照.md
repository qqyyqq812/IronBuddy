## 2026-04-18 15:05 · ⚠️ V3 权重兼容性 smoke 失败（关键发现）

- **做了什么**：跑了 `load_model('models/extreme_fusion_gru.pt', input_size=7)` 验证权重是否可用。

  ```
  Unexpected: fc.weight, fc.bias
  gru.weight_ih_l0 shape [48, 4] (不是期望的 [48, 7])
  ```
