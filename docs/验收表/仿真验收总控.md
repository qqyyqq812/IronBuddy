# IronBuddy 真正极简·终极验收指南

请始终确保你在项目根目录：
```bash
cd /home/qq/projects/embedded-fullstack
```

### 1. 🟢 一键部署与启动 (自动推代码、开后备、开网络)
直接在任意终端运行：
```bash
bash scripts/一键启动全部终端.sh
```
*完成后在浏览器打开 `http://localhost:5000` 即可测试。*

> **注意：关于 UI 为什么看起来没变化？**
> 我们改得非常克制！修改并没有改变主屏幕结构，请你**点击右侧最下方的【设置 (齿轮图标) 】面板卡片**！
> 滑到下面你就能看到全新的 **[本局 MVC 校准]** 大按钮了。

### 2. 🔴 结束与停止
测试完毕关电脑前，一键斩断所有后台，绝不死机卡死：
```bash
bash scripts/一键关闭全部终端.sh
```

### 3. 【无传感器本地拦截强测法】
（当一键启动后大屏幕看着你的时候，新开一个终端敲这段黑客脚本）
```bash
python3 -c '
import time, json
with open("/dev/shm/emg_heartbeat", "w") as f: f.write("1")
print("🔥 假传感器注入中，你屏幕应该会立刻报警 [Ctrl+C 停止]")
try:
    while True:
        data = {"target_rms": 60, "comp_rms": 85, "mdf": 0.8}
        with open("/dev/shm/muscle_activation.json", "w") as f: json.dump(data, f)
        time.sleep(0.05)
except KeyboardInterrupt: pass
'
```
