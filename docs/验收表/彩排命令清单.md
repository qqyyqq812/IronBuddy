## === A 段：深蹲彩排（板端执行）===

### A1. 启动"标准"深蹲模拟器（后台长挂，纯视觉时静默）

```bash
python3 tools/simulate_emg_from_mia.py --label standard
```

> 📌 此时开板、打开 UI、做动作。
> - **纯视觉模式**：终端日志显示 `[SILENT(pure_vision)]`，UI 肌电条为 0
> - 语音切到**视觉+传感**后：日志变 `[label=standard]`，UI 肌电条活起来
> - 做 3–5 个标准深蹲 → 期望 GRU 分类命中 `standard`
> - 完成后按 **Ctrl+C**

### A2. 切"不标准"（半蹲偷懒）

```bash
python3 tools/simulate_emg_from_mia.py --label non_standard
```

> 做 3–5 个半蹲 → 期望分类 `non_standard`。**Ctrl+C**

### A3. 切"代偿"（臀部代偿）

```bash
python3 tools/simulate_emg_from_mia.py --label compensating
```

> 做 3–5 个含代偿动作 → 期望分类 `compensating`。**Ctrl+C**

## === B 段：哑铃弯举彩排（板端执行，含 MVC）===

### B0. 语音切换到弯举

> 对着板端麦克风：
> ```
> "教练" → "切换到弯举"
> ```
> 等 voice 播报 "已切换到弯举" 再继续。

### B1. MVC 校准（关键：语音触发后 3 秒内手动运行脚本）

> 先语音：
> ```
> "教练" → "开始 MVC 测试"
> ```
> 听到 voice 说 **"正在测量，请用最大力量收缩"** 时，立刻在 ssh 终端粘贴：

```bash
python3 tools/simulate_mvc_burst.py
```

> 脚本 5s 后自动退出，voice 会播报 "测试完毕，峰值 800，可以开始正式训练"。
> （峰值数字可通过 `--target-peak 850 --comp-peak 500` 自定义）

### B2. 启动"标准"弯举模拟器

```bash
python3 tools/simulate_emg_from_bicep.py --label standard
```

> 做 3–5 个标准弯举 → 期望分类 `standard`。**Ctrl+C**

### B3. "不标准"弯举（幅度不够）

```bash
python3 tools/simulate_emg_from_bicep.py --label non_standard
```

> 做 3–5 个半程弯举 → 期望分类 `non_standard`。**Ctrl+C**

### B4. "代偿"弯举（摆动借力）

```bash
python3 tools/simulate_emg_from_bicep.py --label compensating
```

> 做 3–5 个含摆动代偿的弯举 → 期望分类 `compensating`。**Ctrl+C**

## 关键设计备忘

| 机制 | 作用 |
|-----|-----|
| 模式闸门 | `pure_vision` 下模拟器发零 → UI 肌电条静默；`vision_sensor` 下发合成波形 |
| 时序对齐 | 模拟器实时读膝角 / 肘角 → 按 phase 从波形池抽样 → ±15% 噪声 |
| MVC 独立 | `simulate_mvc_burst.py` 启动即写 `/dev/shm/emg_calibration.json`，voice 倒计时完可读到 |
| 3 类 label 差异 | standard=golden 原样 / non_standard=整段×0.4 / compensating=target×0.5 + comp×2.0 尖峰 |
